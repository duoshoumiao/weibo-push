# weibo-push
支持hoshino v1

微博关注小插件

微博推送帮助

关注微博 [微博ID]：关注指定微博账号（仅本群生效） 

取消关注微博 [微博ID]：取消关注指定微博账号（仅本群生效） 

查看关注的微博：查看本群已关注的微博账号 

全群关注微博 [微博ID]：所有已加入的群都关注并开启推送（管理员）

微博推送开关 [on/off]：开启或关闭本群微博推送（管理员）
